What It Generates & Design Decisions

ServingRuntime

  • NIM container image + tag
  • GPU resource limits
  • Model volume mount
  • NGC API key env var
  • Liveness/readiness probes

InferenceService

  • Links to ServingRuntime
  • PVC storage reference
  • GPU tolerations
  • fsGroup: 1000 (NIM UID)
  • Autoscaling config

PVC

  • Model cache storage
  • ReadWriteOnce access
  • Configurable size
  • StorageClass override

Additive Only

Never removes or wraps existing NIM paths. All generated resources are new — not replacements.

Disabled by Default

nimServing.<model>.enabled: false in values.yaml. Users opt in explicitly per model.

Secret Handling: Detects existing NGC secrets in the blueprint and references them. Only generates fallback Secret templates when none exist.
